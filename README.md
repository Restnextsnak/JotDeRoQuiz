# JotDeRo Quiz

**모두가 라이어인 역추리 퀴즈쇼**

방송인의 주관적인 취향과 습관을 맞추는 퀴즈 게임. 문제를 보지 못한 채 선택지만 보고 답을 골라야 하며, 틀린 플레이어는 변명을 통해 구제받을 수 있습니다.

## 🎮 게임 플로우

1. **블라인드 선택** (20초): 플레이어들은 문제 없이 4개 선택지만 보고 하나를 선택
2. **문제 공개**: 모두 선택 완료 후 문제 공개, 방장이 정답 지정
3. **변명 타임** (10초): 오답자들이 20자 이내 변명 작성
4. **1:1 채팅**: 방장이 마음에 드는 변명을 선택해 1:1 채팅으로 심문
5. **판결**: 방장이 구제 또는 탈락 결정
6. **다음 라운드**: 최후의 1인이 남을 때까지 반복

## 🎯 주요 기능

- **1.5:7:1.5 레이아웃**: 왼쪽(채팅), 중앙(플레이어 목록), 오른쪽(문제/선택지)
- **실시간 동기화**: Socket.IO 기반 멀티플레이어
- **좋아요 시스템**: 변명에 좋아요를 누르면 플레이어 카드 크기 증가
- **역할 시스템**: 방장, 플레이어, 관전자 구분
- **1:1 채팅**: 방장과 플레이어 간 비공개 대화
- **주관적 질문**: 방송인의 취향/습관/의견을 묻는 15개 질문 제공

## 📦 설치 및 실행

### 로컬 개발

```bash
# 의존성 설치
npm install

# 서버 실행
npm start
```

서버가 `http://localhost:3000`에서 실행됩니다.

### Render 배포

1. GitHub 저장소에 코드 푸시
2. [Render](https://render.com) 대시보드에서 New > Web Service
3. 저장소 연결
4. 설정:
   - **Name**: jotderoquiz (원하는 이름)
   - **Environment**: Node
   - **Build Command**: `npm install`
   - **Start Command**: `npm start`
   - **Instance Type**: Free
5. Create Web Service 클릭

배포 후 `https://your-app-name.onrender.com`에서 접속 가능합니다.

## 🎨 레이아웃 구조

```
┌─────────────────────────────────────────────────┐
│  왼쪽 (1.5)   │     중앙 (7)      │  오른쪽 (1.5) │
│  개인 채팅방  │   플레이어 목록    │   문제/선택지  │
│              │   - 방장 (상단)    │              │
│  1:1 대화    │   - 플레이어들     │   4지선다     │
│  심문        │   - 변명 표시      │   타이머      │
│  판결 버튼   │   - 좋아요 버튼    │              │
└─────────────────────────────────────────────────┘
```

## 📝 게임 규칙 커스터마이징

`questions.json` 파일을 수정하여 질문을 추가/변경할 수 있습니다:

```json
{
  "id": 16,
  "question": "방송인이 가장 자주 먹는 야식은?",
  "options": ["치킨", "피자", "라면", "떡볶이"]
}
```

- 질문은 방송인의 주관적 의견/습관/취향에 관한 것이어야 함
- 정답이 없는 주관식 질문이 재미있음
- 당황스러운 질문일수록 좋음

## 🎭 역할별 기능

### 방장 (Host)
- 게임 시작/다음 라운드 진행
- 문제 공개 후 정답 선택
- 변명 확인 및 1:1 채팅 대상 선택
- 플레이어 구제/탈락 결정
- 즉시 탈락 버튼 (채팅 없이 탈락)

### 플레이어 (Player)
- 선택지만 보고 답 선택
- 오답 시 변명 작성
- 다른 플레이어 변명에 좋아요
- 방장과 1:1 채팅으로 설득

### 관전자 (Spectator)
- 게임 진행 관전만 가능
- 선택, 변명, 채팅, 좋아요 불가
